# TrackBridge Chrome Extension

Manifest V3 extension for Gmail.

## Features

- Sign in to a self-hosted TrackBridge API
- Floating TrackBridge panel on Gmail
- Insert a tracking pixel into the active compose window
- Generate and insert tracked links into the active compose window
- Open the TrackBridge dashboard quickly

## Local installation

1. Open Chrome.
2. Go to `chrome://extensions`.
3. Enable Developer mode.
4. Click Load unpacked.
5. Select the `extension` folder from this repository.
6. Pin TrackBridge, open the popup, enter your TrackBridge URL, and sign in.

Use `https://trackbridge.maranda.io` as the URL for the current production deployment.
